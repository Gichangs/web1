Hypertext markup Laguage (HTML) is the standard markup laguage for creating web pages and web appliactations. HTML (Hypertext Markup Language) is not a programming language, it is a markup language that allows browsers to know how the web pages we see are structured.

This allows developers to program complexly and simply. HTML is made up of elements , which wrap and mark up different parts of each piece of content to display and execute in an appropriate way.
